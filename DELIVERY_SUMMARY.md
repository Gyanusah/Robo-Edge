## 🎨 Customization Points

### Colors
Edit `frontend/tailwind.config.js`

### Company Info
Edit `frontend/src/pages/Home.jsx`, `About.jsx`, etc.

### Google Maps
Update embed URL in `Contact.jsx`

### Google Reviews
Update link in `Testimonials.jsx`

### Admin Credentials
Update in `backend/.env` and run `node seed.js`
